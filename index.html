Topologia em Estrela (Star Layout)

Star Topology (Star Layout)

Topología en Estrella (Diseño en Estrella)

Arquitetura padrão das redes Ethernet modernas. Cada máquina ou CLP possui seu próprio cabo conectado a um switch central.

Standard architecture for modern Ethernet networks. Each machine or PLC has its own cable connected to a central switch.

Arquitectura estándar de las redes Ethernet modernas. Cada máquina o PLC tiene su propio cable conectado a un switch central.

Vantagens

Advantages

Ventajas

  • Se um cabo quebrar, apenas uma máquina isolada cai.
  • Muito fácil de descobrir em qual cabo está o defeito.
  • If one cable breaks, only one isolated machine stops.
  • Very easy to find which cable has the fault.
  • Si un cable se rompe, solo una máquina aislada se detiene.
  • Muy fácil de encontrar qué cable tiene la falla.

Desvantagens

Disadvantages

Desventajas

  • Se o Switch Central quebrar, a fábrica inteira para.
  • Gasta muito mais metros de cabo elétrico.
  • If the Central Switch breaks, the whole factory stops.
  • Uses significantly more meters of electrical cable.
  • Si el Switch Central se avería, toda la fábrica se detiene.
  • Consume muchos más metros de cable eléctrico.

Topologia em Barramento (Bus Layout)

Bus Topology (Bus Layout)

Topología en Bus (Diseño en Bus)

Modelo clássico de redes de campo (Profibus). Todos os equipamentos compartilham em linha o mesmo cabo físico.

Classic fieldbus model (Profibus). All devices share the exact same linear physical cable.

Modelo clásico de redes de campo (Profibus). Todos los equipos comparten en línea el mismo cable físico.

Vantagens

Advantages

Ventajas

  • Grande economia de cabos e infraestrutura metálica.
  • Instalação inicial limpa e linear pelo galpão.
  • Great savings in cables and metallic infrastructure.
  • Clean and linear initial installation through the warehouse.
  • Gran ahorro en cables e infraestructura metálica.
  • Instalación inicial limpia y lineal por la nave.

Desvantagens

Disadvantages

Desventajas

  • Se o cabo principal partir, tudo o que vem depois desliga.
  • Extrema dificuldade para achar conectores frouxos.
  • If the main cable breaks, everything down the line disconnects.
  • Extremely difficult to find loose connectors.
  • Si el cable principal se rompe, todo lo que sigue se desconecta.
  • Extrema dificultad para encontrar conectores flojos.
